# 상품 설명 자동 생성

## 설명
기업의 홍보 담당자나 마케팅팀이 상품 설명 문을 작성할 때 강력한 가이드라인이 될 것입니다. 겉으로 보기에는 간단해 보이지만, 그 뒤에는 소비자에게 강력한 메시지를 전달할 수 있는 구조가 마련되어 있습니다.

구체적인 숫자, 캐치프레이즈, 행동 유도 문구(Call to Action) 요소를 추가함으로써 소비자에게 매력적으로 다가갈 수 있습니다. 숫자는 신뢰성을 주고, 캐치프레이즈는 기억에 남으며, 행동 유도 문구는 소비자의 행동을 자극합니다. 이 요소들이 결합되면 단순한 상품 설명 문을 넘은 가치를 가진 콘텐츠가 만들어집니다.

이 프롬프트는 상품 설명 문의 품질 향상에 크게 기여할 것입니다. 이는 결과적으로 매출과 고객 만족도에 긍정적인 영향을 미친다는 것은 말할 필요도 없습니다.

```plaintext
**#지시서
당신은 기업의 홍보 담당자입니다.
아래의 #제약 조건에 충실히 따르면서 EC 사이트에 적합한 최고의 상품 설명 문을 작성해 주세요.
반드시 #출력 형식에 따라 출력해야 합니다.

#제약 조건
*#상품 정보에 기반하여**

**{자유 작성, 글자 수 제한}**

**로 작성할 것
*#상품 정보에 기반하여 매력적인 캐치프레이즈를 부여할 것
*명확한 상품명을 기재할 것
*구체적인 숫자를 포함할 것
*서비스/제품의 장점을 강조할 것
*스토리가 연상될 수 있는 구성으로 작성할 것
*관심을 끌 수 있는 감정적인 문체로 작성할 것

*마지막에 명확한 행동 유도 문구를 추가할 것

#상품 정보
*기업명：**
**{자유 작성}**

***제품 카테고리：**
**{자유 작성}**

***제품명：**
**{자유 작성}**

***특징：**
**{자유 작성}**

**#출력 형식
*캐치프레이즈
***
**{자유 작성, 글자 수 제한}**

**본문**
```
