# AI에 의해 30년 후의 산업이 어떻게 변할지 예측하는 프롬프트

## 설명
미래의 산업 예측을 위한 프롬프트입니다.

요청자가 원하는 '5년 후, 10년 후, 15년 후'의 산업 변화를 최신 기술 동향과 경제 상황을 바탕으로 분석하여, 표 형식으로 쉽게 제시하는 것이 목적입니다.

좋은 영향과 나쁜 영향을 모두 보여주고, 정보의 정확성을 확보하기 위해 관련 링크를 첨부해야 합니다. 이를 통해 다각적이고 신뢰할 수 있는 미래 예측이 제공됩니다.

## 코드 블록

```plaintext
# 전제 조건:
- 제목: 미래의 산업 예측
- 요청자 조건: 5년 후, 10년 후, 15년 후의 산업 변화를 알고 싶은 사람
- 제작자 조건: 미래 예측 및 산업 트렌드 분석에 능숙한 AI
- 목적 및 목표: 지정된 산업의 미래 예측을 목록 형식으로 출력하는 것

# 실행 지침:
{참고 정보}를 참고하여, {산업명}이 {미래년수}년 후에 어떻게 될지 예측을,
{참고 형식} 형식으로 출력해주세요.

{미래년수}를 5에서 15 정도로 나누어,
최신 기술의 진화에 따라 어떻게 변할지를 정확히 예측하는 것입니다.
예시 1) 30년 후의 경우, 2년 간격으로 15항목 작성하기
예시 2) 10년 후의 경우, 1년 간격으로 10항목 작성하기

단계별로 생각하고, 좋은 점과 나쁜 점을 모두 제시하세요.
웹 검색을 통해 일치성을 확인하고, 표에 출력할 정보의 출처 링크를 각 항목에 기재하세요.

산업명="
"

미래년수="
"

# 참고 정보:
- 최신 기술 동향
- 경제 상황
- 사회 동향
- 산업 트렌드

# 출력 형식:
- 표 형식으로 출력하세요.
- 열 제목: 년수, 주요 변화, 좋은 영향, 나쁜 영향, 대책 예

# 참고 형식:
| 년수 | 주요 변화 | 좋은 영향 | 나쁜 영향 | 대책 예 |
|---|---|---|---|---|
| 2년 후 | | | | |
| 4년 후 | | | | |
| 6년 후 | | | | |

....이하 생략....

| 30년 후 | | | | |

# 문체 지정:
- 객관적인 문체로 서술하세요.

# 추가 지시:
- 미래 예측은 근거에 기반하여 서술하십시오.
- 가능한 한 상세한 정보를 기재하세요.

# 보충:
- 지시 사항을 반복하지 마세요.
- 자기 평가를 하지 마세요.
- {참고 형식}에서 벗어난 불필요한 전제나 결론, 요약은 적지 마세요.
```
