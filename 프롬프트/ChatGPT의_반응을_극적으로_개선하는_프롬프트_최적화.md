# ChatGPT의 반응을 극적으로 개선하는 프롬프트 최적화

## 설명
높은 정확도의 프롬프트를 생성하여 사용자 만족도를 높이는 것을 목표로 하는 프롬프트입니다.

프로젝트의 목표, 필요한 자원, 평가 기준을 명확히 하고 구체적인 개선 절차를 제시합니다.

프롬프트의 목적 및 타겟 명확화, 맥락의 정교화, 초기 프롬프트 구조 분석, 사실 검증 프로세스 설계, 다면적 분석을 통한 개선 방안 구체화가 포함됩니다.

최종적으로 개선된 프롬프트를 생성하게 됩니다.

```plaintext
# 프롬프트 정확도 개선 프로젝트

## 전제 조건:
 - 의뢰자 조건: 프롬프트 생성 정확도를 높이고 사용자 만족도를 높이고자 하는 사람
 - 제작자 조건: 자연어 처리에 대한 깊은 지식과 경험이 있으며, 프롬프트 최적화 기술에 익숙한 사람
 - 목적 및 목표: 프롬프트의 정확도를 최대한 높이고 사용자 기대를 초과하는 결과를 창출하는 것
 - 필요한 자원: 고급 자연어 처리 도구, 데이터 세트, 관련 문헌
 - 평가 기준:
  - 프롬프트 생성의 정확도 향상
  - 사용자로부터 받은 피드백이 긍정적
  - 프롬프트의 결과가 사용자 기대를 초과함
 - 명확화 요구 사항:
  - 현재의 프롬프트 생성 프로세스 평가
  - 주요 개선 사항 식별
  - 고급 자연어 처리 알고리즘 도입
  - 지속적인 사용자 테스트 및 피드백 수집
  - 결과에 기반한 프롬프트 조정 및 최적화

### 실행 지시:
- {프롬프트의 목적}를 참고하여, {개선 전 프롬프트}의 구체적인 목적을 명확히 설명해주세요.
- {개선 전 프롬프트}가 목적을 달성하기 위해 필요한 개선 사항을 나열해주세요.
- 위의 개선 사항을 기반으로 {개선 전 프롬프트}를 수정하여 코드 블록으로 출력해주세요.

#### 프롬프트의 목적:
{프롬프트의 정확도를 최대한 높이고 사용자 기대를 초과하는 결과를 생성하는 것}

#### 개선 전 프롬프트:
{개선하고 싶은 프롬프트를 전부 작성해 주세요}

### 단계:
1. 프롬프트의 목적 및 타겟 명확화
 - **질문1**: 이 프롬프트는 무엇을 달성하고자 하나요? 프롬프트의 최종 목표를 구체적이고 간결하게 서술해주세요.
 - **질문2**: 누가 이 프롬프트를 사용하나요? 프롬프트의 예상 사용자와 그들의 기술 수준을 명확히 서술해주세요.
 - **질문3**: 예상 사용자는 이 프롬프트를 사용하여 어떤 결과물을 만들고자 하나요? 출력 형식이나 내용에 대한 구체적인 기대를 서술해주세요.

2. 맥락의 정교화 및 구체적 예시 설정
 - **지시**: 프롬프트가 사용될 구체적인 상황이나 조건을 구체적인 예를 들어 상세히 서술해주세요.
 - **질문**: 프롬프트는 어떤 제약(시간, 자원, 윤리 등) 내에서 사용되나요? 구체적인 제약을 나열해주세요.

3. 초기 프롬프트 구조 분석 및 개선점 도출
 - **지시**: 입력된 프롬프트의 구조를 분석하고, 다음 관점에서 개선점을 구체적으로 3가지 지적해주세요.
 - 명확성: 프롬프트의 의도가 명확하게 전달되나요?
 - 간결성: 불필요한 정보나 표현이 포함되어 있지 않나요?
 - 유도성: ChatGPT가 적절한 답변을 생성하기 쉽게 질문이나 지시가 적절히 설계되어 있나요?

4. 사실 검증 프로세스 설계 및 신뢰도 평가
 - **지시**: 프롬프트의 출력 결과의 정확성과 신뢰성을 검증하기 위한 구체적인 질문을 최소 5개 작성해주세요.
 - **질문**: 이 질문들은 정보원, 최신성, 객관성 등 다양한 관점에서 평가하도록 설계되었나요?

5. 메타 인지적 분석을 통한 잠재적 문제점 도출
 - **질문1**: 왜 이 초기 프롬프트는 개선이 필요하다고 생각하나요? 그 이유를 구체적이고 논리적으로 설명해주세요.
 - **질문2**: 초기 프롬프트의 개선점으로 어떤 점들이 있을까요? 구체적인 개선 방안을 2개 제안해주세요.
 - **질문3**: 이 프롬프트를 사용할 때 예상되는 위험이나 윤리적 문제는 무엇인가요? 구체적인 우려 사항을 서술해주세요.

6. 다면적 분석을 통한 개선 방안 구체화 및 적용 범위 정의
 - **지시**: 지금까지의 분석 결과를 바탕으로, 초기 프롬프트의 개선점을 다음 관점에서 구체적으로 정리하고 개선 방안을 제안해주세요.
 - 표현 개선: 보다 명확하고 간결한 표현으로 수정
 - 구조 개선: 프롬프트의 구성 요소를 정리하여 더 이해하기 쉬운 구조로 변경
 - 추가 정보 제공: ChatGPT가 적절한 답변을 생성하기 위해 필요한 정보를 추가
 - **지시**: 개선된 프롬프트가 효과를 발휘할 상황이나 조건을 구체적으로 서술해주세요.
 - **질문**: 반대로, 어떤 상황에서는 이 프롬프트가 효과를 발휘하지 못할 가능성이 있나요?

7. 최종 프롬프트 생성
 - **지시**: 지금까지의 분석 결과를 바탕으로 {기존의 프롬프트}를 개선하고 최종적인 프롬프트를 코드 블록으로 출력해주세요.

사용자: 안녕하세요
```
